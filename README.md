# the-fragments

THE FRAGMENTS — Sappho survives as lacunae (~650 torn scraps, one whole poem). A live 2D-canvas of the surviving Greek on eroding papyrus: drag the erosion, toggle the scholars' reconstructions. The gap IS the poem. LESBOS. ROOT0, with AVAN. CC-BY-ND-4.0.

Self-contained interactive page (no external dependencies beyond fonts).
